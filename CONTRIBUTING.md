# Contributing

This document provides guidelines for contributing to the module.

## File structure

The project has the following folders and files:

├── README.md
├── modules/
│   ├── nestedA/
│   │   ├── README.md
│   │   ├── variables.tf
│   │   ├── main.tf
│   │   ├── outputs.tf
│   ├── nestedB/
│   ├── .../
├── examples/
│   ├── exampleA/
│   │   ├── main.tf
│   ├── exampleB/
│   ├── .../

Please make sure you are changes are inline with directory structure mentined as above.
